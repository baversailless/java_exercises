1) Создать перечисление Операция
2) Определить в нем несколько констант, описывающих математическую операцию.
Например, SUBTRACT
3) Определить в нем абстрактный метод abstract int action(int x, int y).
4) Для каждой константы реализовать этот метод. Внутри метода должна быть реализована операция, соответствующая названию и смыслу этой константы
5) Создать аннотацию для класса Фигура.
6) Аннотация должна указывать считается ли площадь фигуры геометрически верным образом
7) Она также должна указывать какую операцию из перечисления Операция следует применить для расчета площади фигуры (условно, Операция Ы). В этой операции есть метод action
8) Клиентский код в методе А прежде, чем вызвать метод расчета площади фигуры, должен проинспектировать наличие аннотации. Если она указана для данного класса, и если она указывает, что площадь фигуры в методе этого класса считается геометрически неверным способом, то клиентский код должен вызвать метод action(), который указан аннотацией в поле Операция. Так как у метода action два параметра, а класс Фигура предоставляет только одно измерения, то допускается при вызове этого метода  action() в качестве второго аргумента передать любое число (например, 10)
9) Если аннотация указана, и она указывает, что площадь фигуры в методе этого класса считается геометрически верным способом, то клиентский код должен, вывести площадь обычным образом.
10) Если аннотация не указана, то клиентский код должен вывести площадь обычным образом
